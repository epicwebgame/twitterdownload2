# Twitter-Video

App to download videos from twitter with multiple features.

| Main Screen |  
| ----------- |
|<img src="images/twitter.jpeg" width="300" height="500">|

## About

- Download Videos from Twitter
- Auto-Listen: when the option is activated, the app recognizes a video links from twitter then downloads it
- Day/Dark mode: You can switch between Day mode and Dark mode
